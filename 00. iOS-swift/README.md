# iOS Swift Study

Swift 문법 공부한 내용을 정리하는 레포입니다.

## 목표

- Swift ( SwiftUI / MVVM ... 등 ) 기본 문법 익히기

## 진행 단계

Swift 기초 
( 변수, 함수, 옵셔널, 배열/딕셔너리, 옵셔널과 guard, struct / class / enum, Codable )

## 폴더 구조

- `01-swift-basics1`  
  Swift 변수, 함수, if, 배열, 딕셔너리
  
- `01-swift-basics2`  
  옵셔널과 guard
  
- `01-swift-basics3`  
  struct / class / enum
  
- `01-swift-basics4`  
  Codable (설정/메타데이터 저장에 필요)

- `01-swiftUi-basics1`  
  SwiftUI 뷰 기초 (View / @State / 바인딩)

- `01-mvvm-basics1`  
  MVVM 기초 문법 (ObservableObject / @Published / @StateObject)
  
- `01-save-basics1` 
  저장 관련 문법 (UserDefaults / FileManager + Codable)
  
- `01-timer-basics1` 
  타이머 & 비동기 기초 (자동 넘김 준비)
  
- `01-protocol-basics1` 
  프로토콜 & 의존성 주입 기초
